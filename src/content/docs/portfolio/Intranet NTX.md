# NTX Connect - Sistema de Gestión Corporativa

Proyecto de intranet corporativa para centralizar documentación, identidad y procesos internos de Noritex.

## Impacto de negocio

- Unificación de gestión documental pública y privada por departamentos.
- Centralización de identidad de colaboradores con sincronización de fuentes heterogéneas.
- Mejora de coordinación interdepartamental mediante un portal único.

## Arquitectura

- Backend monolito modular con Django y API REST.
- Frontend SPA con Next.js desacoplado.
- Integraciones con Microsoft Graph API y almacenamiento cloud.

## Funcionalidades clave

- Motor de solicitudes y aprobaciones con flujos por jerarquía.
- Gestor documental conectado a SharePoint/OneDrive.
- Autenticación corporativa SSO.
- Buzón de sugerencias interno.

## Stack tecnológico

- Python, Django, Django REST Framework
- TypeScript, Next.js, React, Tailwind CSS
- PostgreSQL
- Docker
- AWS S3, Microsoft Graph API, Azure AD

## Resultado

La plataforma se convirtió en el núcleo operativo interno para colaboración y acceso seguro a información corporativa.
