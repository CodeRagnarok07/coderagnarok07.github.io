# UPISA - Distribuidora internacional de cárnicos

Proyecto de rediseño e internacionalización del sitio oficial de UPISA para mejorar experiencia de usuario y ampliar alcance comercial internacional.

## Objetivos

- Modernizar la interfaz y navegación del sitio.
- Internacionalizar contenido a múltiples idiomas.
- Permitir gestión autónoma de contenido mediante un CMS propio.
- Incorporar funcionalidades interactivas para negocio y usuarios.

## Implementación

- Internacionalización en español, inglés, ruso y chino.
- Optimización SEO por idioma.
- Desarrollo de administrador de contenidos con Python y Django.
- Módulos de catálogo filtrado, recetas y calculadora de requerimientos.

## Stack tecnológico

- Python, Django
- React.js
- Tailwind CSS, Sass
- JavaScript
- Hosting en cPanel

## Resultado

El sitio quedó preparado para crecimiento internacional, con mejor gestión editorial y una experiencia más clara para usuarios finales.
