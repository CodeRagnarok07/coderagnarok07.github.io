# Operadores Aritmeticos

```python
x = x + 2  # suma"
x = x - 2  # Resta"
x = x * 2  # Multiplicacion"
x = x / 2  # Divicion"
x = x % 2  # Modulo"
x = x // 2 # Divicion sin float, sirve para redondear al minimo"
x = x ** 2 # Potencia"
x = -x     # negated
x = +x     # unchanged
```
